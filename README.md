## ReadBird 

### Used Library
- react-router-dom
- redux
- redux-toolkit
- react-hook-form
- react-modal
- sweet-alert
- dayjs
- react-dropdown
- react-icons/all-file
- react-calendar
- react-window 

### Directory Structure
```
├─assets    // 이미지 or 폰트
├─components    // 컴포넌트
├─constants     // 공통적으로 사용되는 상수
├─hooks     // 커스텀 훅
├─pages     // 페이지
├─store     // 상태관리
├─styles    // css
└─utils     // 정규표현식 패턴이나 공통 함수 등
```
